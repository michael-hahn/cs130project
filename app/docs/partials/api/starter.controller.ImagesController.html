ImagesController
function in module starter

Description

Images controller of the app.

Usage

ImagesController();

Methods

  • endEvent()

    Ends the event so that it still exists, but no user can add pictures to it

  • eventUserList()

    Brings up a list of all users associated with the event

  • filter(userEmail, timeLower, timeUpper)

    Filters the photos in the event based on the user and/or a date range

    Parameters
    ParamTypeDetails
    userEmailstring

    The email of the user to filter photos by

    timeLowerstring

    The lower bound of the time to filter by

    timeUpperstring

    The upper bound of the time to filter by

  • reactivateEvent()

    Reactivates the event so that users can add pictures to it again

  • unfilter()

    Removes the filter on photos (displaying them all)

  • upload(type)

    Uploads a photo to the current event (either a new photo or an existing one from the user's photo gallery)

    Parameters
    ParamTypeDetails
    typeint

    The user's selection of how to upload a picture (either taking a new one or selecting one from their photo gallery)

  • viewPhoto(photoData, index)

    Brings up a page to view the selected photo

    Parameters
    ParamTypeDetails
    photoDataObject

    The data of the photo that the user selected

    indexint

    The index of the photo the user selected
